# Python-Code-Class-12

A collection of Python programs written during Class 12 preparation for **CBSE Board Exams** and **CUET CS Domain Exam**.

## 📥 Installation

Make sure [Git](https://git-scm.com/) is installed, then run:

```bash
git clone https://github.com/kounsaiswaroop/Python-Code-Class-12.git
cd Python-Code-Class-12
```

## 🚀 How to Run

Make sure Python 3 is installed, then:

```bash
python3 filename.py
```
